CSS Grid intro:

  • Grid is used for two-dimensional layouts (both rows and columns).
  • CSS Grid is the preferred option rather than float layout.
  • Why it's fast and clean:
    • Minimal CSS: Just a few lines to achieve a fully responsive layout.
    • Flexibility: The grid automatically adjusts to the screen size.
    • Efficiency: No media queries are needed for responsiveness, making the code cleaner and more maintainable.
  • CSS Grid allows you to create flexible, responsive layouts without relying on heavy CSS frameworks like Bootstrap or Foundation.

Grid Container

  1. Grid Container
    • Definition: The element that holds the grid items.
    • Usage: To define an element as a grid container, apply display: grid; or display: inline-grid;.
  2. Grid Items
    • Definition: The direct children of a grid container are automatically treated as grid items.
    • Placement: These items are positioned within the grid based on the rules you define.
  3. Grid Template Columns/Rows
    • Definition: These properties define the number of columns and rows, as well as their sizes.
    • Usage: You can define them in fixed units (px, em, etc.), percentages, or even fractions (fr).
  4. Grid Gaps
    • Definition: The space between grid items
    • Usage: grid-gap, grid-row-gap, and grid-column-gap are used to specify the gaps between rows and columns.
  5. Grid Lines
    • Definition: The lines that separate the rows and columns in the grid.
    • Usage: You can refer to these grid lines when positioning items.
